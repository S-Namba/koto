”合奏（セッション）ができる箏（こと）のＷＥＢアプリ”を目指して作っています。

＜現在の仕様＞
・弦に見立てた四角形を指でタッチしてください。そうすると、音が出ます
・動作はややもっさりしています。なるべく早く反応できるようにしたいです
・タッチとクリックに対応しています
・ストローク（って言うんでしょうか。名前は知りませんがジャラーンってやつ）に対応したいと思っています
・現在はピアノの音を使っています。箏の音ではありません
・箏の画像は本物の箏の写真を使っています。とある所で撮影したものを使っています
・学校の授業等で使われる場合は、必ず授業前に実機検証することをお勧めします

2023/02/23「現在はMP3で音を出していますが、MIDIベースにすればもっと反応を速くできるんじゃないかと考えています」→MIDIベースにしても大して反応を速くできないようなので、対応しないことにしました

2023/02/23「タッチにのみ対応しています。クリックには対応していません。近々対応したいです」→タッチとクリックに対応しました
